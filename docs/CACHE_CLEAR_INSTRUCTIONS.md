# Browser Cache Clearing Instructions

The JavaScript files have been FIXED on disk (all smart quotes removed).
However, your browser is serving OLD CACHED versions of the files.

## Solution: Clear Browser Cache Completely

### For Chrome/Edge:
1. Press **Ctrl + Shift + Delete**
2. Select "Cached images and files"  
3. Time range: "All time"
4. Click "Clear data"
5. **Close the browser completely**
6. Reopen and navigate to the page

### For Firefox:
1. Press **Ctrl + Shift + Delete**
2. Check "Cache"
3. Time range: "Everything"
4. Click "Clear Now"
5. **Close the browser completely**
6. Reopen and navigate to the page

### Alternative Quick Fix:
1. **Close ALL browser windows completely**
2. Run Launch.bat again
3. The fresh browser session should load the corrected files
